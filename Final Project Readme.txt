(How to run program in Docker)
- open project from command line "media"
- go into the Dockerfile directory(Directory inside media)
- Once inside type "docker-compose up -d"
- Go to IP address specified

(How to run program in PHPstorm)
-Open Project "media"
-Go to the commandline or in project terminal type "php artisan serve"
-Go to the following address in any internet browser.
Defult port should be (http://127.0.0.1:8000)
